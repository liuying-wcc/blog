# 🦋 流萤 · Firefly Blog

「就算终将化为灰烬，也要在燃尽之前照亮夜空」✨

一个以 **崩坏：星穹铁道 · 流萤** 为主题的二次元风格个人博客。

## 🎨 特性

- **🦋 流萤主题设计** — 深空色背景 + SAM 荧光绿主题色
- **✨ 萤火虫粒子动画** — 页面漂浮着发光的萤火虫
- **🔥 5 个选项卡** — 主页、文章、项目、相册、关于
- **📝 打字机效果** — 流萤经典台词轮播
- **🖼️ 相册灯箱** — 点击图片放大查看
- **📱 完全响应式** — 适配手机、平板和桌面

## 📂 项目结构

```
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式文件（流萤暗黑主题）
├── js/
│   └── main.js         # 交互脚本
├── README.md
├── _config.yml         # GitHub Pages 配置
└── .gitignore
```

## 🚀 部署到 GitHub Pages

```bash
git remote add origin https://github.com/liuying-wcc/blog.git
git push -u origin main
```

然后在 GitHub 仓库 Settings → Pages 中，选择 `main` 分支的 `/ (root)` 作为源。

## 🛠️ 自定义指南

- **修改个人信息**：编辑 `index.html` 中的内容
- **修改主题色**：编辑 `css/style.css` 中的 `:root` 变量
- **添加文章**：在 `<section id="blog">` 中添加新的 `article` 卡片
- **更多效果**：在 `js/main.js` 中调整动画参数
